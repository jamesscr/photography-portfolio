### üèóÔ∏è Installation <a name="installation"></a>

1. Clone the repo

```sh
 git clone https://github.com/jamesscr/photography-portfolio/.git
```

Run the following command:

1. npm run build or yarn build will build your Tailwind CSS.
2. npm start or yarn start will start your Express server.
3. npm run dev or yarn dev will start your server with nodemon for development.
4. npm run prod or yarn prod will build your CSS and then start the server, suitable for production.

5. Run the Project in a Server

6. Change it with your own photos, about me and contact info.

7. Publish it using your preferred hosting platform.
